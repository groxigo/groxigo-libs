# @groxigo/components

Composite UI components built from `@groxigo/ui-elements`.

## Overview

This package provides higher-level, composite components that combine multiple ui-elements to create complete UI patterns. These components include business logic and are ready to use out of the box.

## Installation

```bash
npm install @groxigo/components
```

## Dependencies

This package depends on:
- `@groxigo/ui-elements` - Primitive building blocks
- `@groxigo/tokens` - Design tokens

## Usage

```typescript
import { ProductCard, SearchBar, FormField } from '@groxigo/components';
```

## Components

Components will be added here as they are developed. Examples:

- **ProductCard** - Product display card with image, title, price, and actions
- **SearchBar** - Search input with icon and clear button
- **FormField** - Complete form field with label, input, and error handling
- **Modal** - Dialog/modal component
- **DatePicker** - Date selection component

## 8px Grid System

All components follow the 8px grid system for consistent spacing and sizing.
